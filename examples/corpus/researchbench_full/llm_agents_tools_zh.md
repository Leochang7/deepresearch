# Function Calling and Tool Use in LLM Agents

## 函数调用机制

大语言模型的函数调用（Function Calling）是一种允许模型与外部工具和API交互的机制。模型在推理过程中可以输出结构化的JSON参数，用于调用预定义的函数。

## 工作原理

函数调用流程包括三个阶段。首先，开发者定义可用的工具模式（tool schema），包括函数名称、描述和参数规范。其次，模型根据用户请求判断是否需要调用工具，并生成结构化的参数。最后，系统执行函数调用并将结果返回给模型进行后续推理。

## 厂商支持

OpenAI 在 GPT 系列模型中率先引入了原生函数调用支持。随后 Anthropic 在 Claude 模型中实现了类似的工具使用（tool use）能力。两家公司的实现都支持并行函数调用和嵌套调用模式。

## Toolformer 自监督学习

Toolformer 是 Meta 提出的一种通过自监督学习实现工具使用的方法。与基于提示的方法不同，Toolformer 在训练过程中自主学习何时以及调用哪个工具。该方法不需要手动编写工具使用示例，而是通过token预测任务自动发现工具调用时机。

## 与提示方法的区别

基于提示的工具调用需要在系统提示中手动描述工具使用规则，而 Toolformer 将工具使用能力直接嵌入模型参数中。这使得 Toolformer 在推理效率和工具选择准确性方面具有优势。
