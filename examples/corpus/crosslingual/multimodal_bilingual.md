# Multimodal Alignment / 多模态对齐

CLIP uses contrastive learning to align image and text representations. Matching image-caption pairs are pulled closer together, while mismatched pairs are pushed apart.

Vision-language models map different modalities into a shared embedding space so that images and text can be compared or retrieved with the same similarity function.

多模态模型通常把图像、文本等不同模态编码到共享向量空间。对比学习通过拉近匹配样本、推远不匹配样本来建立跨模态语义对齐。
